Detailed Description

The LobbyAgent class is intended to manage the communication with the Lobby. The role of the Lobby, writtent in Java, is to route the clients to the server. The server receives from the Lobby the number of clients to wait for, and starts the game after a corresponding number of players are connected. This class accesses PongShared objects, thus inherits the class Concurrent.

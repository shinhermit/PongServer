 intended to terminate the thread which executes this worker We call Concurrent classes, classes which need to access the pbjects in the class PongShared. Inside the methods of all Concurrent classes, there are break points, that allow to stop the execution safely, and then send the signal finishedSignal(), which should be connected to the slot quit() of the thread which executes this worker class.

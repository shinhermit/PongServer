Detailed Description

The Lockable class is intended to ensure access to shared objects. Shared objects are gathered in the class PongShared as static members. The main purpose of this class is to avoid the need of passing mutexes to every method that need to access the shared object. However the addition of the class Concurrent makes this class less usefull, we kept it for concenience.
